# 🚀 NextGen Selenium Automation Framework

## 📌 Project Overview
NextGen Selenium Framework is a Hybrid Test Automation Framework developed using:

- Selenium WebDriver
- Java
- TestNG
- Maven
- Page Object Model (POM)
- Data Driven Testing (Excel)
- Log4j Logging
- Screenshot Capture
- TestNG Listeners

---

## 🏗 Framework Architecture
- BaseTest Setup
- Page Classes (POM Design)
- Utilities Layer
- Data Providers (Excel Driven)
- Listener Reporting
- Config Management

---

## 🧪 Features
✅ Data Driven Testing using Excel  
✅ Page Object Model Design Pattern  
✅ Screenshot Capture on Failure  
✅ Explicit Wait Utilities  
✅ Logging using Log4j2  
✅ Maven Dependency Management  
✅ TestNG Parallel Execution Ready
